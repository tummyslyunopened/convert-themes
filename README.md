# convert-themes

Convert kitty themes to alacritty themes with ease.

## Usage
1. Clone this repo 
```git clone git@github.com:tummyslyunopened/convert-themes.git
cd convert-themes
ls
```
1. copy kitty .conf theme files to the kitty-themes subdirectory

1. uv run convert.py
1. view results:
```
cd alacritty-themes
cat test.toml
```
